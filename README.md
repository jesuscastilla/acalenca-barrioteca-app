# acalenca-barrioteca-app
La aplicación web progresiva (PWA) que acompaña a la barrioteca de Acalencá, en Salobreña. Creada desde 0 por /jesuscastilla

EJECUCIÓN:
1.- npm run dev
2.- npm run build
3.- open index.html

Tenemos un servidor propio, autogestionado, seguro y confidencial, para más información visitad
https://pelotxo.synology.me/acalenca
